What is PSI?
PSI is an language for defining program's behaviour, its designed to be both human and machine friendly while being able to be translated into other codes like assembly or low-level codes 

module program;
entry main;

required #print;

constant $str @message "Hello, World!";

function sum $int $int $int;

test {
    add %result !0 !1;
    ret %result;
}

main {
    mov $int %x 7;
    mov $int %y 5;
    call sum %result %x %y;
    sys #print @message; 
}
